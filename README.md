## :fire:  Configurando o Ambiente de Desenvolvimento

- ### **Pré-requisitos**

  - É **necessário** possuir o **[Node.js](https://nodejs.org/en/)** instalado na máquina
  - Também, é **preciso** ter um gerenciador de pacotes, se você já tem o Node.js, então o **[NPM](https://www.npmjs.com/)** já vai estar disponível.
  - Por fim, é **essencial** seguir as intruções para instalar o **[Android Studio e Android SDK](https://developer.android.com/studio/)**

1. Faça um clone :

```sh
  $ git clone https://github.com/JimmyBastos/gomarketplace.git
```

2. Executando a Aplicação:

```sh
  # Execute o comando abaixo para instalar as dependencias do projeto
  $ yarn

  # Inciciar o servidor de desenvolvimento
  $ yarn start

  # Executar app
  $ yarn android/ios
```
